# Keywords

The following keywords are reserved and may not be used as identifiers:

* `and`
* `andalso`
* `case`
* `else`
* `end`
* `fun`
* `gen`
* `hdatatype`
* `in`
* `if`
* `let`
* `list`
* `module`
* `nil`
* `not`
* `of`
* `orelse`
* `ref`
* `sdatatype`
* `sw`
* `then`
* `type`
* `unsw`
* `val`
